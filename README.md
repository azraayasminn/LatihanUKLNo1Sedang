Program Java ini menghitung:
1. Faktorial dari bilangan positif.
2. Volume tabung berdasarkan jari-jari dan tinggi.

## Cara Kerja
- Masukkan bilangan = tampil hasil faktorial.
- Masukkan jari-jari dan tinggi = tampil volume tabung.

## Rumus
- Faktorial: n! = 1 × 2 × ... × n  
- Volume tabung: π × r² × t
